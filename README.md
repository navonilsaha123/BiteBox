# 🍔 BiteBox – Online Food Ordering System

## 📖 Overview
BiteBox is a full-stack online food ordering platform that allows users to browse restaurants, explore menus, add food items to their cart, and place orders seamlessly. The application provides a fast, secure, and responsive experience for customers while simplifying food ordering and order management.

---

## ✨ Features
- 🔐 User Authentication (Login & Registration)
- 🍕 Browse Restaurant Menus
- 🔍 Search & Filter Food Items
- 🛒 Shopping Cart
- 📦 Order Management
- 👤 User Profile
- 📱 Fully Responsive Design
- ⚡ Fast and User-Friendly Interface

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/BiteBox.git
```

### Navigate to the Project

```bash
cd BiteBox
```

### Install Dependencies

**Frontend**
```bash
cd client
npm install
```

**Backend**
```bash
cd ../server
npm install
```

### Run the Application

**Backend**
```bash
npm start
```

**Frontend**
```bash
cd ../client
npm start
```
